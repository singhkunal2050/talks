# Cursor Meetup Mumbai - November 2025

## Building a Blog Publisher MCP using Cursor

### Setup

```bash
npm install
```

### Run

```bash
npm run dev
```

This will start the Slidev server at http://localhost:3030

### Build

```bash
npm run build
```

### Export to PDF

```bash
npm run export
```

### Navigation

- Use arrow keys or space to navigate
- Press `o` for overview mode
- Press `d` to toggle dark mode
- Press `f` for fullscreen
